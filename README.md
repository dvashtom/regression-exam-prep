# 📊 Regression Exam Prep - אפליקציית הכנה למבחן ברגרסיה

## התקנה והרצה

```bash
cd regression_exam_prep
pip install -r requirements.txt
streamlit run app.py
```

## מבנה הפרויקט

```
regression_exam_prep/
├── app.py                  # אפליקציית Streamlit ראשית
├── pages_extra.py          # קוד נוסף לעמודי תרגול ומעקב
├── data/
│   └── exam_analysis.json  # נתוני ניתוח מבחנים ומודל חיזוי
├── requirements.txt
└── README.md
```

## תכונות

- **🏠 דף ראשי** - סיכום מהיר, מבנה צפוי, טיפים
- **🔮 מודל חיזוי** - הסתברויות הופעת נושאים, מפת חום, ניתוח מבחני עבר
- **📚 נוסחאות** - כל הנוסחאות המפתח לפי נושא עם LaTeX
- **✍️ תרגול** - שאלות בסגנון המבחן עם פתרונות
- **📈 מעקב** - מעקב התקדמות אישי

## הערה
האפליקציה מבוססת על ניתוח 15+ מבחנים מהשנים 2011-2025 בקורס "מודלים של רגרסיה ליניארית" של ד"ר יקיר ברצ'נקו.
